## 基于vue的外卖web APP

> 从jQuery跨越到MVXX框架学习花了很长一段时间，因为这是一个观念上的转变问题，因为熟悉了指令式的jq开发一下跨入框架未免有些反应不过来，所以在长期的学习摸索有了一个感性认识之后，为了巩固所学于是才有这个demo，说它是demo不过分，因为没有使用vuex之类的来构建大型应用，组件间通信都是evenbus。而且模拟后端的数据过于简洁，不过总算是满足vue平时业务代码开发需求的。

## Technology stack
vue + vue-router + webpack + ES6 + vue-resource + stylus + better-scroll

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build

```
